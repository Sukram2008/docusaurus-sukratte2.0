# ☢️💸 | Wirtschaft zurücksetzen

Befehl: **`/economy destroy`**

Optionen:
- **`confirm:`** [Optional]  
  Trage `True` oder `False` ein.

<u>Beschreibung:</u>  
 Damit wird das komplette Economy System zurückgesetzt.

<u>Berechtigungen:</u>  
  Nur eingerichtete Nutzer können den Befehl nutzen.
